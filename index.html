Abstract

Varun Machhale Kumar

M.S. Electrical & Computer Engineering, Purdue University

Applying machine learning to neuroscience, ranging from clinical applications to understanding neural circuits. Current work builds end-to-end software pipelines for high-density Neuropixels recordings, and investigates how theta oscillations support memory encoding in mouse models, and replicates these mechanisms in artificial neural networks using predictive coding.

Prior work covered epileptic seizure classification from interictal EEG (journal, patent) and medical image segmentation (bladder, cardiac) pipelines. In a previous life, worked with electronics, capacitors, inductors, resistors, transistors and robotics.
