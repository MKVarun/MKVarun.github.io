Abstract

Varun Machhale Kumar

M.S. Electrical & Computer Engineering, Purdue University

Applying machine learning to neuroscience ranging from clinical applications to understanding neural circuits. Current work builds end-to-end software pipelines for high-density Neuropixels recordings, and a focus on how theta oscillations support memory encoding in mice models and how we can replicate them in neural networks using predictive coding.

Prior work covered epileptic seizure classification from interictal EEG (journal, patent) and medical image segmentation (bladder, cardiac) pipelines. In a previous life, worked with electronics, capacitors, inductors, resistors, transistors and robotics.
